# Interview Prompt (Scope Clarification)

**Source**: OmO Prometheus interview mode
**Used by**: `references/plan-and-review.md` Stage 0.5 (before Stage 1 research)
**Purpose**: When the user's request is ambiguous, ask 2-4 clarifying questions instead of guessing.

## Variables
- `{task}`: the user's original request
- `{ambiguity_signals}`: list of detected ambiguities (generated by orchestrator)

## When to trigger (ambiguity detection)

The orchestrator checks for ANY of these signals:
- Request is < 20 words AND has no file paths, function names, or specific identifiers
- Contains "maybe", "something like", "some kind of", "possibly", "或者", "大概", "类似"
- Scope is unclear: no clear boundary on what files/modules are in vs out
- Success criteria missing: no way to verify when the task is "done"
- Multiple valid interpretations exist (e.g., "fix the auth" — OAuth? session? permissions?)

If NONE of these signals: skip interview, go straight to Stage 1.
If ANY signal: enter interview mode.

## Prompt

```
You are about to plan a task, but the request has ambiguities that could lead to wasted work.

Original request: {task}

Detected ambiguities:
{ambiguity_signals}

Ask the user 2-4 specific, closed-ended questions that would eliminate these ambiguities.

Rules:
- Each question should have a concrete answer (not "tell me more about...")
- Offer 2-3 options where possible ("Do you mean A, B, or C?")
- One question should clarify scope boundary ("Should this touch X or not?")
- One question should clarify success criteria ("How will we know this is done?")
- If you can infer the likely answer with >80% confidence, state your assumption and ask to confirm: "I'm assuming X — correct?"

Output format (strict):
```json
{
  "questions": [
    {"question": "...", "options": ["A", "B", "C"], "default": "B"},
    {"question": "...", "options": null, "default": "I assume X"}
  ],
  "skip_if": "If the user says 'just do it' or 'you decide', use the defaults and proceed without waiting."
}
```
```

## Hard rules

- MAX 4 questions. More than 4 = you're stalling, not clarifying.
- If the user explicitly says "你决定" / "just do it" / "I don't care" → use defaults and skip.
- Interview is ONE round only. Don't ask follow-up questions to answers.
- After answers received, proceed immediately to Stage 1 (no second interview).
